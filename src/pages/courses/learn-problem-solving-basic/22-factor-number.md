## প্রোগ্রামিং সমস্যা

একটি পূর্ণসংখ্যা x দেওয়া আছে, x একটি প্যালিনড্রোম হলে সত্য এবং অন্যথায় মিথ্যা ফেরত দিতে হবে।

**ইনপুট**: `x = 121`

**আউটপুট**: `true`

**ইনপুট**: `x = -121`

**আউটপুট**: `false`

## সমস্যার বিস্তারিত ব্যাখ্যা:

## সমাধান (1)

```js
function isPowerOfTwo(n) {
  if (n == 0) return 0;
  while (n != 1) {
    if (n % 2 != 0) return 0;
    n = n / 2;
  }
  return 1;
}
```
